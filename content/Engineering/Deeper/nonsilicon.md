# Non-Silicon Computing

In the past, there were some chips made with gallium arsenide (such as those in the [Cray-3](https://en.wikipedia.org/wiki/Cray-3)) though it shouldn't be confused with [gallium nitride](https://en.wikipedia.org/wiki/Gallium_nitride) (GaN) which are getting common use not for logic, but for power electronics.
